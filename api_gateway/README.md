# API Gateway

## Зона ответственности
API Gateway служит единой точкой входа для всех внешних запросов. Он маршрутизирует запросы к соответствующим сервисам, выполняет аутентификацию и авторизацию пользователей.

## Границы сервиса
- Получает запросы от пользователей.
- Перенаправляет запросы в `User Service`, `Statistic Service`, `Post and Comment Service`.
- Обрабатывает ошибки и отвечает клиенту.
- Не хранит данные, а только управляет маршрутизацией.
