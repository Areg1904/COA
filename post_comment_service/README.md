# Post and Comment Service

## Зона ответственности
Post and Comment Service отвечает за создание, редактирование и удаление постов и комментариев.

## Границы сервиса
- Управляет постами и комментариями пользователей.
- Хранит данные в PostgreSQL.
- Публикует события о лайках, просмотрах и комментариях в Kafka.
- Не управляет пользователями или статистикой.
