# Statistic Service

## Зона ответственности
Statistic Service отвечает за сбор, обработку и хранение событий, связанных с активностью пользователей.

## Границы сервиса
- Получает события (просмотры, лайки, комментарии) из Kafka.
- Обрабатывает данные и сохраняет их в ClickHouse.
- Предоставляет API для получения статистики.
- Не обрабатывает создание постов, комментариев или управление пользователями.
