# User Service

## Зона ответственности
User Service отвечает за управление пользователями. Он обрабатывает регистрацию, авторизацию и управление профилями пользователей.

## Границы сервиса
- Создание и управление учетными записями.
- Аутентификация и авторизация пользователей.
- Хранение данных о пользователях в PostgreSQL.
- Не управляет постами, комментариями или статистикой.
